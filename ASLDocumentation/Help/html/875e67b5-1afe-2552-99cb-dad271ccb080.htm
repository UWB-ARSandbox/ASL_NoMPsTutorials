LobbyManager Methods

Methods
  NameDescription
Connect
Connects a user to a game using GameSparks auto-matchmaking ability based upon the room name that the programmer specified in the Unity Editor for the QuickConnect script if they are using QuickConnect. Otherwise is moves onto the next step of manually connecting players
Disconnect
Disconnects the user from the specified match
DisplayPanel
Set active the passed in panel while deactivating all others.
GetMatchShortCodes
Get all of the match Short Codes for this project (these blueprints for matches are created here https://portal2.gamesparks.net/games/G376367AmCEW/config/multiplayer/matches, thus these short codes don't change during the game and therefore this function can be called from start.
HostRoom
Create a new match making request, hosting a match based upon the dropdown menu option
JoinMatch
Requests to join the selected match
OnApplicationQuit
Disconnect the user when they quit. Only used when quitting on the lobby scene.
OnAuthentication
Gets called every time a player gets authenticated with the GameSparks API for ASL. We use it to update the user's username and connection status as well as move them to the next panel.
OnMatchFound
Triggers when a match is found either through hosting or through selecting a match. Sets the lobby screen variables
OnMatchUpdate
Triggers when a match is found either through hosting or through selecting a match. Sets the lobby screen variables
OnRegistration
Gets called every time a player gets registered with the GameSparks API for ASL. We use it to update the user's username and connection status as well as move them to the next panel
ResetRoomView
Destroy the match buttons and reset the original positioning of the match buttons so the refreshed list contains up to date matches, placed correctly
SearchForHostedRooms
Search for and display the possible matches the player could join
SetupHostDropDownMatchOptions
Add the matches have been created in the GameSparks API/browser (https://portal2.gamesparks.net/games/G376367AmCEW/config/multiplayer/matches) to our dropdown menu for the user to select what match they want to host. These matches were discovered by GetMatchShortCodes() called in our start function
Start
Start is called before the first frame update
